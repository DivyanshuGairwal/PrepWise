# PrepWise

PrepWise is an AI-powered interview preparation platform that generates personalized interview questions based on a candidate's resume and a target job description.

The goal is to help students, freshers, and job seekers prepare for interviews with questions tailored to their projects, skills, and target role.

## Features

- Upload Resume (PDF)
- Paste Job Description
- AI-powered analysis
- Technical interview questions
- Resume-specific questions
- Behavioral questions
- HR questions
- Clean and responsive UI
- Export interview kit

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- OpenRouter API
- PDF Parsing

## How It Works

1. Upload your resume.
2. Paste a job description.
3. Generate your personalized interview preparation kit.
4. Practice using AI-generated questions.
